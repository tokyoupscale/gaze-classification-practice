# обработка датасета

## описание датасета

**источник:** [dataset](https://data.mendeley.com/datasets/vy4n28334m/1)

**характеристики:**

- количество изображений: 7500
- количество участников: 12
- классы направления взгляда: 5
- снимки на камеру мобильного телефона

## метки классов

| класс | описание | количество изображений |
|-------|----------|------------------------|
| `up` | взгляд вверх | 1500 |
| `down` | взгляд вниз | 1500 |
| `left` | взгляд влево | 1500 |
| `right` | взгляд вправо | 1500 |
| `straight` | взгляд прямо | 1500 |

## Предобработка данных

### 1. организация структуры

```bash
dataset/
|- up/
|- down/
|- left/
|- right/
|- raight/
```

### 2. разделение данных

- **train:** 75%
- **validation:** 10%
- **test:** 15%

### 3. аугментация

Применяемые трансформации для обучающей выборки:

- ресайз 224х224
- нормализация

##  анализ датасета

### 1. баланс классов

- датасет сбалансирован, каждый класс содержит одинаковое количество изображений (1500), что исключает проблему class imbalance.

### 2. качество данных

- изображения сняты в различных условиях освещения

### 3. проблемы и ограничения

- все изображения сделаны на мобильную камеру - может отличаться от веб-камер
- потенциально малая вариативность фона
- отсутствие данных для экстремальных углов взгляда

## вивади

датасет вполне подходит для задачи классификации направления взгляда, сбалансированность классов и достаточное количество примеров позволяют обучить модель
