# классификация направления взгляда

система классификации направления взгляда пользователя в режиме реального времени с использованием вебкамеры и CNN модели

## описание проекта

проект создан фор фан хз

**основные возможности:**
- классификация направления взгляда на 5 классов: up, down, left, right, straight
- fastapi сервер
- графический интерфейс на streamlit с поддержкой веб-камеры через webrtc (realtime streaming)
- кастомная CNN модель на torch


## howto run

### 1. установка зависимостей

```bash
# 1.
pip install -r requirements.txt
```

```bash
# 2.
python api.py
```
Сервер запустится на http://localhost:5252

```bash
# 3.
streamlit run app.py
```

### 2. использование

1. запустите api сервер (python api.py)
2. запустите streamlit app (streamlit run app.py)
3. дайте доступ к веб-камере в браузере
4. система определяет направление взгляда и отобразит результат
